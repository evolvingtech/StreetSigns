14.Examples of HyperSign Projects:

 

There are many uses for this signsource. It was developed to make multimedia a truly useful tool for creative access for Deaf Learners. It was also created to provide a lively source for Sign Language information.

Some example of the HyperSign environment include:

Star-Crossed Studies: An interactive project based on the Sign Language production of Romeo and Juliet.

The Guernica Project: A project combining text, pictures, Sign Language and animation exploring Picasso's painting.

"Thank you, M'am": An interactive project supporting Langston Hughes' idiomatic text with American Sign Language videoclips.
