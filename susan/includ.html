12.How to use StreetSigns for Sign Language Inclusion:

 

If you want to use the Signs in this project to include in your lessons you need to use a program that plays QuickTime movies. Many presentation, word processing, and authoring programs can play digital movies. Check the program documentation to be sure.

 

Open up your program. Choose the command that inserts digital movies. Navigate through your system to find the Media folders that contain the Sign Movies. Find the movie by name and open it up into your program!
